# TrustFlow

A mobile fintech onboarding and KYC system built with Flutter.

## Overview
TrustFlow demonstrates production-ready onboarding flows designed for Nigerian fintech use cases, focusing on reliability, failure handling, and compliance-aware UX.

## Features (In Progress)
- Progressive onboarding flow
- BVN/NIN capture
- Document verification
- Face capture
- Failure-first UX
- State persistence

## Tech Stack
- Flutter
- BLoC (State Management)
- Clean Architecture
